# 🎮 Naruto Shippuden: Ultimate Ninja Storm Generations | Tradução PT-BR

![Status](https://img.shields.io/badge/Status-Conclu%C3%ADdo-brightgreen)
![Plataformas](https://img.shields.io/badge/Plataformas-PS3%20%7C%20Xbox%20360%20%7C%20PC-orange)

Tradução completa e independente para o título **Naruto Shippuden: Ultimate Ninja Storm Generations**. Este projeto traz a localização total para o Português Brasileiro, abrangendo menus, legendas e interface para consoles e emuladores.

---

## 🛠️ Compatibilidade Total

O projeto segue uma estrutura idiomática (organizada conforme os padrões da comunidade) para funcionar perfeitamente em:

- **Consoles Reais:**
    - **PlayStation 3:** Compatível com CFW e HEN.
    - **Xbox 360:** Compatível com consoles desbloqueados (RGH/JTAG/Exploit).
- **Emuladores (PC):**
    - **RPCS3:** Emulador de PS3.
    - **Xenia:** Emulador de Xbox 360.

## 🚀 Como Instalar

As instruções detalhadas e os arquivos estão disponíveis no [Site Oficial](https://jeankasoki.github.io/naruto-storm-generations-ptbr/).

### 📂 Instruções de Arquivo
- **Patch de Tradução:** Se você já possui o jogo original E QUER APENAS A TRADUÇÃO, baixe o patch e substitua a pasta `PS3_GAME` na raiz. O mesmo vale para a versão de Xbox 360, substituindo a pasta `data`
